# Maggie & Noah - Engagement Announcement

A modern, elegant engagement announcement website for Maggie and Noah.

**Live Site:** https://maggienoah.vercel.app

## Features

- **Photo Slideshow** - Full-screen hero with Ken Burns zoom effect and auto-transitioning photos
- **Responsive Design** - Optimized for desktop, tablet, and mobile
- **Scroll Animations** - Fade-in reveal effects as content enters viewport
- **Photo Gallery** - Three engagement photos displayed in a strip section
- **Wedding Details** - Placeholder cards for date, time, and venue (TBD)
- **RSVP Form** - Interactive form with animated confirmation
- **Modern Navigation** - Fixed nav with scroll-aware styling

## Tech Stack

- HTML5
- CSS3 (custom properties, flexbox, grid, animations)
- Vanilla JavaScript
- Google Fonts (Playfair Display, Cormorant Garamond, Montserrat)

## Deployment

Hosted on Vercel with automatic deployments from GitHub.

```bash
# Manual deploy
vercel --prod
```

## Local Development

Simply open `index.html` in a web browser.
