# Maggie & Noah's Wedding Announcement

A beautiful landing page announcing the wedding of Maggie and Noah.

## Features
- Responsive design
- Modern and elegant layout
- RSVP section
- Countdown timer to the big day

## Setup
Simply open `index.html` in a web browser to view the announcement.

- HTML5 for structure
- CSS3 for styling
- JavaScript for interactive elements
